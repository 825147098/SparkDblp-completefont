# sparkdblp-font

### 项目说明
通过大数据技术加快数据查询速度，提供简单查询接口，并提供用户友好的UI。

### 前端技术栈
  1. Vue
  2. ElementUI
  3. axios
  4. vue-router
  5. Vuex
  6. vue-cli3
  7. ECharts
